## Featured Projects

- [powershell-offboarding](https://github.com/JamesTill-dev/Offboarding-Script)
  PowerShell script automating hybrid AD and M365 user offboarding — account disable, group removal, license revocation, shared mailbox conversion, session revocation, and out-of-office configuration. Built for a production environment.

- [homelab](https://github.com/JamesTill-dev/homelab)
  Documented configuration for a self-hosted media and networking stack on a Synology NAS. Includes Docker Compose files for Plex, the *arr suite, Gluetun VPN routing, and Pi-hole — with runbooks, troubleshooting notes, and backup strategy.

- [sip-tracker](https://github.com/JamesTill-dev/sip-tracker)
